**한 줄 요약**: Microsoft Build 2026에서 마이크로소프트는 AI 에이전트가 Windows를 직접 제어하고 다단계 업무를 자율 처리하는 미래 비전을 제품 출시로 구체화했다.

### 핵심
- **Microsoft Scout**: Windows 11+·macOS 12+ 대상 첫 'Autopilot' 에이전트 — 상시 실행되는 독립 AI로 Frontier 프리뷰 프로그램에서 오늘부터 이용 가능
- **GitHub Copilot App**: 여러 에이전트를 병렬로 지시할 수 있는 데스크톱 네이티브 앱 프리뷰 출시
- **Microsoft IQ**: GitHub Copilot·Foundry·Copilot Studio 전반을 연결하는 공통 컨텍스트 레이어 — Work IQ(업무 지식), Fabric IQ(데이터), Web IQ(웹 그라운딩) 3종으로 구성되어 GA
- **Windows Development Skills**: WinUI3 기반 네이티브 앱 전체 생성 기능 GA — 에이전트가 더 적은 토큰으로 품질 높은 출력 생성
- GitHub Copilot에 디버깅·프로파일링·병합 충돌 해결 자동화 추가

### 왜 중요한가
마이크로소프트는 Copilot을 '부조종사'에서 '자율 파일럿'으로 격상시키고 있다. Scout처럼 항상 켜져 있는 에이전트가 실제 운영 환경에 도입되면, 소프트웨어 개발 방식 자체가 달라진다. GitHub Copilot에 이어 Windows 자체가 에이전트 플랫폼이 되는 전환점이다.
