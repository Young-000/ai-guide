**한 줄 요약**: Anthropic이 Claude Agent SDK·`claude -p`·GitHub Actions 사용량에 별도 월정 크레딧을 부과하는 계획을 시행 당일인 6월 15일 전격 보류했다.

### 핵심
- 당초 계획: 플랜별 월 크레딧 추가 **Pro $20 / Max 5x $100 / Max 20x $200**, 표준 API 요금으로 차감
- 5월 13일 발표 → 6월 15일 시행 예정 → **시행 당일 철회**
- 현재 상태: Agent SDK, `claude -p`, GitHub Actions 사용량은 기존 구독 한도에서 그대로 소화
- 철회 배경: ① 개발자 커뮤니티 반발, ② OpenAI의 API 가격 인하 검토 소식에 따른 경쟁 압력, ③ IPO 준비 국면에서 개발자 이탈 리스크 최소화

### 왜 중요한가
구독 기반 AI 플랫폼에서 에이전트 자동화 사용량을 별도 과금하는 건 업계 전반의 수익화 숙제다. Anthropic의 마지막 순간 철회는 개발자 기반을 희생하지 않고 수익을 늘려야 하는 딜레마를 잘 보여준다. 재시행 여부와 시점이 추후 주목된다.
