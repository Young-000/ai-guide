**한 줄 요약**: Anthropic의 2단계 로봇 실험에서 Claude Opus 4.7이 인간 보조 없이 로봇독 소프트웨어 작업을 완료하는 속도가 인간 팀 대비 최소 10배, 공통 과제 평균 37배에 달했다.

### 핵심
- **실험 내용**: 참가 팀이 로봇독에 자체 컴퓨터를 연결·센서 데이터(영상·라이다)를 활용해 소프트웨어를 직접 개발, 공을 회수
- Opus 4.7(단독): 지난해 인간 팀이 완료한 모든 과제에서 **최소 10배** 빠른 완료; 4개 공통 과제 평균 **37배** 빠름
- 코드 대부분 첫 시도에 바로 작동 — 인간 팀의 반복 디버깅 과정 생략
- 235,000명 이상의 400,000개 세션 데이터 분석 기반
- **한계**: 실시간 시각 피드백 기반 폐루프 정밀제어는 아직 실패 — 실제 공 회수 달성 못 함

### 왜 중요한가
이번 결과는 AI가 계획·코드 생성 영역에서 이미 인간 속도를 압도하고 있음을 보여주는 통제된 실험이다. 다만 실시간 지각-행동 피드백이 필요한 물리적 제어는 여전히 한계다. '얼마나 빠른가'보다 '어디서 막히는가'를 명확히 드러낸 점이 이 연구의 핵심 기여다.
