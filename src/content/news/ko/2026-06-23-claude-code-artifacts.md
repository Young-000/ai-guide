**한 줄 요약**: Anthropic이 Claude Code의 작업 결과물을 실시간 업데이트되는 공유 HTML 페이지로 변환하는 Artifacts 기능을 Team·Enterprise 플랜에 베타 공개했다.

### 핵심
- Claude Code 세션 작업이 **브라우저에서 바로 열리는 자동 업데이트 웹페이지**로 게시됨 — 팀원이 URL 하나로 작업 진행 상황을 실시간 확인 가능
- 주요 활용 사례: **PR 워크스루, 인시던트 대응 페이지, 팀 대시보드, 체크리스트** — 회의 없이 비동기 협업 지원
- 각 Artifact는 **단일 HTML 파일, 최대 16 MiB** — 외부 네트워크 요청을 차단하는 엄격한 CSP 적용으로 보안 사고 방지
- 조직 내 **비공개 공유**(org-only) 및 **버전 히스토리** 지원; Claude Code CLI·데스크톱 앱에서 생성

### 왜 중요한가
AI 코딩 에이전트가 "코드 생성"을 넘어 팀 운영 인텔리전스로 진입하는 신호다. 코드뿐 아니라 데이터 소스까지 연결된 라이브 페이지를 Claude가 자율 업데이트하면서, 인간이 리뷰하고 방향을 잡는 비동기 협업 모델이 현실화되고 있다.
