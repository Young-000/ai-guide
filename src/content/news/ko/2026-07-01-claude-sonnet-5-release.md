**한 줄 요약**: Anthropic이 6월 30일 Claude Sonnet 5를 공개하며 Free·Pro 기본 모델로 승격했다. Opus 4.8에 근접한 성능을 기존 대비 낮은 비용으로 제공한다.

### 핵심
- **기본 모델 승격**: Free·Pro 플랜 전면 교체, Max·Team·Enterprise도 즉시 이용 가능
- 성능은 Opus 4.8에 근접 — 전작 Sonnet 4.6 대비 추론·도구 사용·코딩·에이전틱 작업 전 영역 향상
- 출시 가격: 입력 **$2/백만 토큰**, 출력 **$10/백만 토큰** (8월 31일까지, 이후 각 $3·$15)
- Claude, Claude Code, API, GitHub Copilot 동시 적용
- 브라우저·터미널 도구 사용 및 장기 자율 실행을 소형 모델 비용 수준에서 실현

### 왜 중요한가
Sonnet 5는 몇 달 전까지 대형 모델 전용이던 에이전틱 워크플로우를 중간급 가격대로 끌어내렸다. 기업이 에이전트를 대규모 배포할 때 비용·성능 균형점이 이번 모델로 사실상 재설정된다.
