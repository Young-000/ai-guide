**한 줄 요약**: Moonshot AI가 2.8조 파라미터 오픈웨이트 모델 Kimi K3를 출시해 프론트엔드 코딩 벤치마크에서 Claude Fable 5를 제쳤다.

### 핵심
- 파라미터 수 2.8조 개 — 현재까지 공개된 오픈웨이트 모델 중 최대 규모
- Frontend Code Arena 1위(1,679점), Kimi K2.6 대비 17계단 상승(18위 → 1위)
- SWE Marathon·Program Bench 1위; Terminal Bench 2.1에서 GPT-5.6 Sol에 0.5점 차 2위
- API 가격: 캐시 히트 $0.30/M, 캐시 미스 $3/M, 출력 $15/M 토큰; 오픈웨이트 공개는 7월 27일 예정

### 왜 중요한가
미국의 반도체 수출 규제 속에서도 중국 AI 모델이 코딩 특화 벤치마크에서 서구 최전선 모델을 추월한 것은 의미가 크다. 7월 27일 오픈웨이트 공개 시 역대 최대 규모의 모델을 누구나 내려받아 파인튜닝할 수 있게 되어 오픈소스 생태계에 상당한 파급력이 예상된다.
