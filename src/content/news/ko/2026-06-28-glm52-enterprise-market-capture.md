**한 줄 요약**: 미국 수출 통제로 Fable 5·Mythos 5 접근이 막히고 GPT-5.6도 극소수 파트너에만 제공되는 상황에서, Zhipu AI의 GLM-5.2가 기업 시장 공백을 빠르게 채우고 있다.

### 핵심
- GLM-5.2: 6월 13일 MIT 라이선스 완전 오픈웨이트 공개 — 다운로드·파인튜닝·자체 배포 가능, 별도 승인 불필요
- BridgeBench Reasoning 1위 주장, 코딩 벤치마크에서 GPT-5.5를 1/6 비용으로 초과 달성, 속도 300 토큰/초
- API 가격: 입력 $1.40/M, 출력 $4.40/M — 동급 미국 프론티어 모델($8~$15/M) 대비 80% 저렴
- 100만 토큰 컨텍스트, 멀티모달, 에이전틱 워크로드 지원
- 리스크 이중 구조: API 사용 시 데이터가 Zhipu 중국 인프라를 경유; 가중치 자체 호스팅 시 이 위험 제거
- CNBC: Fable 5·GPT-5.6 접근이 막힌 기업들이 현재 GLM-5.2 도입 평가를 적극 진행 중

### 왜 중요한가
미국 수출 통제는 중국의 프론티어 AI 접근을 늦추려는 목적이었지만, 부작용으로 글로벌 기업 시장에서 중국 모델의 경쟁 공간을 열어줬다. Fable 5나 Mythos 5를 구매할 수 없는 기업들은 다음 선택지를 즉시 평가한다. GLM-5.2의 MIT 오픈웨이트 공개는 이미 다운로드된 모델의 확산을 사실상 막을 수 없음을 의미한다. 폐쇄형 API 수출 통제는 가능하지만, 전 세계 서버에 이미 배포된 오픈웨이트 모델의 사용을 막는 것은 별개의 문제다.
