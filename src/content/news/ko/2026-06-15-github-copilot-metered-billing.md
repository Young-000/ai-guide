**한 줄 요약**: GitHub Copilot이 기존 '프리미엄 요청 건수' 방식을 폐지하고, 토큰 소비량에 연동된 'AI Credits' 종량제로 과금 체계를 전면 개편했다.

### 핵심
- **6월 1일 전환**: 월 구독자는 자동 전환. 연 구독자는 플랜 갱신 시 전환.
- **AI Credits**: 1 Credit = 0.01달러. 플랜별 월 포함량 — Pro 1,500 / Pro+ 7,000 / Max 20,000 크레딧.
- **기업 플랜**: Business 계정 1인당 월 1,900 크레딧, Enterprise는 3,900 크레딧 기본 제공. 2026년 9월 1일까지 한시적으로 각각 3,000/7,000 크레딧으로 상향.
- **비용 구조**: 사용 모델·입력/출력/캐시 토큰 수에 따라 차등 과금. 고사양 모델 사용 시 예상 외 추가 비용 발생 가능.

### 왜 중요한가
개발자가 사용하는 AI 모델과 컨텍스트 크기에 따라 요금이 달라지는 '진짜 종량제'가 업무 도구에 본격 도입되면서, 기업 IT팀은 Copilot 사용량을 적극 모니터링해야 하는 상황이 됐다. 비용 예측성이 낮아진 만큼 사용 패턴 최적화와 모델 선택 전략 수립이 새로운 운영 과제로 부상했다.
