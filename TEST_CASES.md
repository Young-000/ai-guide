## 6. 알려진 제한사항

1. **홈페이지 검색**: `features` 필드는 검색 대상에 포함되지 않음
   - 예: "코딩" 검색 시 Cursor가 나오지 않음 (tagline에 "코딩" 없음)
   - 개선 권장사항: 검색 범위에 features 추가

2. **빠른 검색 버튼 "무료"**: 텍스트 검색으로 작동하여 결과 없음
   - 개선 권장사항: 무료 버튼은 pricing.free 필터로 동작하도록 변경
