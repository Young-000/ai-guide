# AI Guide - PRD

> v2.0 | 최초 작성: 2026-02-13 | 개편: 2026-02-17

## 비전

> **"AI를 처음 만나는 모든 사람이 5분 안에 자신에게 맞는 AI를 찾고, 바로 활용할 수 있는 세상"**

## 프로젝트 목적

AI를 전혀 모르는 일반 대중부터 초급자까지, **"어떤 AI를 써야 할지 모르겠다"** 는 문제를 해결하는 **AI 온보딩 플랫폼**.
사용자의 상황/목적/수준을 파악하여 맞춤형 가이던스를 제공하고, 실제로 AI를 쓰기 시작할 때까지 안내한다.

## 타겟 사용자 (4 페르소나)

| 페르소나 | 설명 | AI 경험 |
|---------|------|---------|
| **직장인 (32세)** | AI 활용 지시받은 마케팅 팀원 | 이름만 아는 수준 |
| **대학생 (22세)** | 과제/발표에 AI 활용하고 싶은 학생 | ChatGPT 기초 사용 |
| **크리에이터 (28세)** | 작업 효율화가 필요한 프리랜서 디자이너 | Canva AI 약간 |
| **자영업자 (45세)** | 상품 설명/마케팅에 AI 도입하고 싶은 쇼핑몰 대표 | 거의 없음 |

## 핵심 기능

1. **스마트 온보딩** - 3~5단계 설문으로 사용자 파악 → 맞춤 AI 추천
2. **개인화 추천 엔진** - 규칙 기반 매칭으로 최적 AI 도구 1~3개 추천
3. **실습형 가이드** - 단계별 따라하기 + 프롬프트 복사 + 결과 확인
4. **활용 사례 라이브러리** - Before/After 실제 사례 (직업별/상황별)
5. **진행 추적** - 레벨/XP 시스템으로 학습 동기 부여
6. **도구 비교 & 카탈로그** - 기존 기능 유지 + 제휴 링크 통합
7. **용어 사전 & 트렌드** - 기존 기능 유지

|--------|-------|------------|

| 프리미엄 콘텐츠 (프롬프트 팩, 코스) | Phase 3~ | 50~100만원 |

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | Next.js 14.2 (App Router) |
| 언어 | TypeScript + React 18 |
| 스타일링 | Tailwind CSS |
| 데이터 | 정적 JSON (Phase 1~2) → Supabase (Phase 3+) |
| 분석 | GA4 + 커스텀 이벤트 |
| 배포 | Vercel |
| 테스트 | Jest + Testing Library → Vitest (Phase 2 마이그레이션) |

## 로드맵 요약

| Phase | 목표 | 사이클 수 |
|-------|------|----------|
| **Phase 1: 기반 구축** | MVP. 온보딩→추천→가이드 핵심 루프 | 5 |
| **Phase 2: 핵심 확장** | 콘텐츠 확장 + SEO + 수익 최적화 | 5 |
| **Phase 3: 성장** | 사용자 계정 + Supabase + 프리미엄 | 5 |
| **Phase 4: 스케일** | B2B + 성능 최적화 + 자동화 | 4 |

- 초기 로딩 2초 이내 (LCP < 2.5s)
- SEO 최적화 (Next.js SSG + 구조화 데이터)
- 모바일 반응형 디자인 (모바일 퍼스트)
- Core Web Vitals 통과
- Vercel 자동 배포
