## 5. 알려진 제한사항

### 5.1 기능적 제한

1. **홈페이지 검색 범위**
   - 현재: `name`, `tagline`, `description` 필드만 검색
   - 제한: `features` 필드는 검색 대상에 포함되지 않음
   - 예시: "코딩" 검색 시 Cursor가 나오지 않음 (tagline에 "코딩" 없음)

2. **빠른 검색 버튼 "무료"**
   - 현재: 텍스트 "무료"를 검색하여 결과 없음
   - 권장: `pricing.free` 속성으로 필터링하도록 변경

### 5.2 개선 권장사항

| 우선순위 | 항목 | 설명 |
|---------|------|------|
| P1 | 검색 범위 확대 | features 필드 포함 |
| P2 | 무료 필터 개선 | pricing.free 기반 필터 |
| P3 | SEO 메타 태그 | 각 페이지별 메타 태그 추가 |
| P3 | 이미지 최적화 | 도구 아이콘 이미지화 |
